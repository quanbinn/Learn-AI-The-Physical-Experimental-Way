- 把下面的这段python代码拷贝到这个页面“In [ ]:”右侧的空白栏中， 然后单击上方的按键“运行”。

```python
import numpy as np
np.arange(16)
B = 2**np.arange(16)

A = np.array( [1,0,1,1,0,0,0,1,0,1,0,1,1,0,1,1])
print(A.dot(B))
print(A.dot(2**np.arange(16)))
```
